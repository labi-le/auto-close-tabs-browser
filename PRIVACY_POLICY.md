# Privacy Policy

**Tab Lifecycle Timer** Chrome Extension

**Effective Date:** June 17, 2026

## Overview

Tab Lifecycle Timer is designed to close inactive browser tabs automatically. This extension is built to work locally in your browser and does not collect personal data.

## Information We Do Not Collect

This extension does **not** collect, store externally, transmit, or sell any of the following:

- Personal information
- Telemetry or analytics data
- Browsing history
- Cookies
- Passwords
- Form data
- Device identifiers

## Information Stored Locally

All extension data is stored **only on your device** using `chrome.storage.local`. This includes:

- Timeout settings
- Whitelist entries
- Protected tab IDs
- Trash bin of closed tabs
- RAM counter
- Locale preference
- Theme preference

No external server receives this data.

## Network Requests

The extension does not send user data over the network. The **only** external network request occurs when the optional **Gachi** theme is enabled, which loads a decorative background video from `gachiradio.com`. This is media content only and is not used for data collection.

## Security and Rendering

The extension uses safe DOM APIs for rendering and avoids unsafe HTML injection patterns.

If this policy changes, the updated version will be published with a new effective date.
